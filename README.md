# CLI for client managnement 
- Golang
- MongoDB
  

```bash

NAME:
   Clients - A simple CLI program to manage your clients

USAGE:
   main [global options] command [command options] [arguments...]

COMMANDS:
   Add, a     Add a client to the list
   all, l     List all clients
   read, r    Read a client by id
   edit, e    Update a client
   delete, d  
   help, h    Shows a list of commands or help for one command

GLOBAL OPTIONS:
   --help, -h  show help (default: false)
```
